# Architecture Decision Records (ADRs) — SmartGate

ADRs documentam as decisões arquiteturais significativas tomadas no projeto SmartGate,
incluindo contexto, alternativas consideradas e trade-offs aceitos.

## Índice

| ADR | Título | Status |
|-----|--------|--------|
| [ADR-001](./ADR-001-offline-first-edge.md) | Arquitetura Offline-First no Edge | ✅ Aceita |
| [ADR-002](./ADR-002-event-driven-architecture.md) | Event-Driven Architecture + Saga Pattern | ✅ Aceita |
| [ADR-003](./ADR-003-cqrs-auditoria.md) | CQRS na Camada de Auditoria | ✅ Aceita |

## Formato adotado

Baseado no formato de Michael Nygard (2011), com campos:
`Status · Data · Autores · Contexto · Decisão · Alternativas · Consequências · Critério de Revisão`

> Referência: [Documenting Architecture Decisions — Michael Nygard](https://cognitect.com/blog/2011/11/15/documenting-architecture-decisions)
